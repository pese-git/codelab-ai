# TUI Клиент

> Руководство по использованию терминального интерфейса CodeLab.

## Обзор

TUI (Terminal User Interface) клиент — основной способ взаимодействия с CodeLab. Построен на базе [Textual](https://textual.textualize.io/) и обеспечивает богатый интерфейс прямо в терминале.

## Запуск клиента

### Подключение к серверу

```bash
# Подключение к локальному серверу
codelab connect

# Подключение к удаленному серверу
codelab connect --host 192.168.1.100 --port 8765
```

### Автономный режим (Local)

```bash
# Запуск сервера и клиента вместе
codelab
```

## Структура интерфейса

```
┌─────────────────────────────────────────────────────────────┐
│  Header Bar (статус подключения, информация о сессии)       │
├──────────────┬──────────────────────────────────────────────┤
│              │                                              │
│   Sidebar    │              Chat View                       │
│  (сессии,    │        (история сообщений)                   │
│   файлы,     │                                              │
│   план)      │                                              │
│              │                                              │
│              ├──────────────────────────────────────────────┤
│              │           Tool Panel                         │
│              │    (результаты выполнения)                   │
├──────────────┴──────────────────────────────────────────────┤
│                    Prompt Input                              │
├─────────────────────────────────────────────────────────────┤
│  Footer Bar (горячие клавиши)                               │
└─────────────────────────────────────────────────────────────┘
```

## Панели интерфейса

### Header Bar

Отображает:
- Статус подключения к серверу (🟢 Connected / 🔴 Disconnected)
- Имя текущей сессии
- Количество сообщений в чате

### Sidebar

Боковая панель с тремя вкладками:

| Вкладка | Описание |
|---------|----------|
| **Sessions** | Список сессий, создание новой, переключение |
| **Files** | Файловое дерево проекта |
| **Plan** | План агента с задачами |

Переключение вкладок: `Ctrl+Tab` / `Ctrl+Shift+Tab`

### Chat View

Основная область отображения:
- Сообщения пользователя (справа)
- Ответы агента (слева)
- Tool calls с результатами
- Запросы разрешений (inline)

### Tool Panel

Отображает:
- Выполняемые команды терминала
- Изменения в файлах (diff)
- Результаты выполнения инструментов

### Prompt Input

Поле для ввода сообщений агенту:
- Многострочный ввод (`Shift+Enter`)
- Отправка (`Enter`)
- Slash-команды (`/help`, `/clear`)

## Горячие клавиши

### Навигация

| Клавиша | Действие |
|---------|----------|
| `Tab` | Переключение фокуса между панелями |
| `Ctrl+B` | Фокус на Sidebar |
| `Ctrl+S` | Фокус на список сессий |

### Управление сессиями

| Клавиша | Действие |
|---------|----------|
| `Ctrl+N` | Создать новую сессию |
| `Ctrl+J` | Следующая сессия |
| `Ctrl+K` | Предыдущая сессия |

### Работа с чатом

| Клавиша | Действие |
|---------|----------|
| `Enter` | Отправить сообщение |
| `Shift+Enter` | Новая строка |
| `Ctrl+L` | Очистить чат |
| `Ctrl+R` | Повторить последний промпт |
| `Ctrl+C` | Отменить текущий запрос |

### Инструменты

| Клавиша | Действие |
|---------|----------|
| `Ctrl+T` | Открыть лог терминала |

### Общие

| Клавиша | Действие |
|---------|----------|
| `Ctrl+P` | Command Palette |
| `Ctrl+T` | Переключить тему (dark/light) |
| `Ctrl+\`` | Открыть терминал |
| `Ctrl+H` или `?` | Открыть справку |
| `Ctrl+Q` | Выход |

## Command Palette

Command Palette (`Ctrl+P`) — быстрый доступ к командам:
- `/new` — новая сессия
- `/clear` — очистить чат
- `/mode` — переключить режим
- `/help` — справка

## Темы

CodeLab поддерживает две темы:
- **Dark** — тёмная тема (по умолчанию)
- **Light** — светлая тема

Переключение: `Ctrl+T` или через Command Palette.

## Streaming текст

Ответы агента отображаются в реальном времени с typing effect. Во время генерации ответа показывается индикатор мышления (ThinkingIndicator).

## Работа с сессиями

### Создание сессии

1. Нажмите `Ctrl+N` или кнопку "New" в Sidebar
2. Введите имя сессии (опционально)
3. Начните диалог с агентом

### Переключение сессий

- Используйте `Ctrl+J` / `Ctrl+K` для перелистывания
- Кликните на сессию в Sidebar

### Загрузка истории

Сессии автоматически сохраняются и восстанавливаются при следующем запуске.

## Разрешения

При запросе агентом доступа к файлам или терминалу появляется диалог:

```
┌────────────────────────────────────────┐
│  🔒 Запрос разрешения                  │
│                                        │
│  read_text_file: src/main.py           │
│                                        │
│  [Allow]  [Allow All]  [Deny]          │
└────────────────────────────────────────┘
```

Варианты ответа:
- **Allow** — разрешить единоразово
- **Allow All** — разрешить для этого типа операций
- **Deny** — отклонить запрос

## Slash-команды

| Команда | Описание |
|---------|----------|
| `/help` | Показать справку |
| `/clear` | Очистить чат |
| `/new` | Создать новую сессию |

## Файловое дерево

Панель Files в Sidebar показывает структуру проекта:

- 📁 Папки можно раскрывать/сворачивать
- 📄 Файлы можно открыть для просмотра
- Изменённые файлы отмечаются индикатором

## План агента

Панель Plan показывает:
- Текущие задачи агента
- Статус выполнения (⏳ In Progress, ✅ Done, ❌ Failed)
- Подзадачи и их прогресс

## Поддержка темной/светлой темы

TUI клиент автоматически адаптируется к настройкам терминала.

## Советы

1. **Используйте Tab** для быстрой навигации между панелями
2. **Ctrl+C отменяет** долгие операции агента
3. **Ctrl+T** показывает детальный вывод терминала
4. **Allow All** ускоряет работу для доверенных операций
