# Горячие клавиши

Справочник клавиатурных сокращений TUI клиента CodeLab.

## Глобальные сочетания

Работают в любом контексте приложения:

| Клавиша | Действие | Описание |
|---------|----------|----------|
| `Ctrl+Q` | Выход | Закрыть приложение |
| `Ctrl+N` | Новая сессия | Создать новую сессию |
| `Ctrl+B` | Фокус на sidebar | Переключиться к списку сессий |
| `Ctrl+S` | Фокус на сессии | Переключиться к списку сессий |
| `Ctrl+H` | Справка | Открыть контекстную справку |
| `Ctrl+P` | Command Palette | Открыть палитру команд |
| `Ctrl+T` | Переключить тему | Переключение dark/light темы |
| `Ctrl+\`` | Открыть терминал | Показать вывод терминала |
| `?` | Горячие клавиши | Показать список всех горячих клавиш |
| `Tab` | Цикл фокуса | Переключение между элементами интерфейса |

## Навигация по сессиям

| Клавиша | Действие | Описание |
|---------|----------|----------|
| `Ctrl+J` | Следующая сессия | Выбрать следующую сессию |
| `Ctrl+K` | Предыдущая сессия | Выбрать предыдущую сессию |
| `Ctrl+Tab` | Следующая вкладка | Переключить вкладку sidebar вперёд |
| `Ctrl+Shift+Tab` | Предыдущая вкладка | Переключить вкладку sidebar назад |

## Работа с чатом

| Клавиша | Действие | Описание |
|---------|----------|----------|
| `Ctrl+L` | Очистить чат | Очистить текущий чат |
| `Ctrl+R` | Повторить | Повторить последний prompt |
| `Ctrl+C` | Отмена | Отменить текущий prompt |

## Поле ввода

| Клавиша | Действие | Описание |
|---------|----------|----------|
| `Ctrl+Enter` | Отправить | Отправить сообщение |
| `Ctrl+↑` | История назад | Предыдущий prompt из истории |
| `Ctrl+↓` | История вперёд | Следующий prompt из истории |

## Окно терминала

| Клавиша | Действие | Описание |
|---------|----------|----------|
| `Escape` | Закрыть | Закрыть модальное окно |

## Просмотр файлов

| Клавиша | Действие | Описание |
|---------|----------|----------|
| `Ctrl+F` | Поиск | Открыть строку поиска |
| `Enter` | Следующее | Перейти к следующему совпадению |
| `Shift+Enter` | Предыдущее | Перейти к предыдущему совпадению |
| `Escape` | Закрыть | Закрыть просмотрщик файла |

## Запросы разрешений

| Клавиша | Действие | Описание |
|---------|----------|----------|
| `A` | Разрешить | Разрешить операцию |
| `R` | Отклонить | Отклонить операцию |
| `Escape` | Отмена | Отменить запрос |

## Модальные окна

| Клавиша | Действие | Описание |
|---------|----------|----------|
| `Escape` | Закрыть | Закрыть модальное окно |
| `Enter` | Подтвердить | Подтвердить выбор (где применимо) |

## Советы по навигации

### Быстрое переключение между сессиями

Используйте `Ctrl+J` и `Ctrl+K` для быстрой навигации по сессиям без переключения фокуса.

### История команд

В поле ввода используйте `Ctrl+↑` и `Ctrl+↓` для навигации по ранее введённым командам.

### Контекстная справка

Нажмите `Ctrl+H` для получения справки, соответствующей текущему элементу интерфейса.

### Полный список

Нажмите `?` для просмотра полного списка горячих клавиш в приложении.
