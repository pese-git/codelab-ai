# 2.6 — Исправление `asyncio.Lock` в `GlobalPolicyManager`

**Приоритет:** 🟡 Средний  
**Оценка:** 3 часа  
**Файл:** `src/codelab/server/protocol/handlers/global_policy_manager.py`

---

## Проблема

`asyncio.Lock()` создаётся на уровне класса при его определении:

```python
class GlobalPolicyManager:
    _instance: GlobalPolicyManager | None = None
    _lock = asyncio.Lock()   # ← создаётся при импорте модуля
```

`asyncio.Lock` привязывается к event loop, существующему в момент создания. В тестах с `pytest-asyncio` каждый тест получает **новый** event loop, а блокировка остаётся привязана к старому. В Python 3.9 это вызывало явную ошибку, в 3.10+ — deprecated warning, но поведение по-прежнему непредсказуемо.

Дополнительно: singleton `_instance` не сбрасывается между тестами → тесты зависят от порядка выполнения.

---

## Решение

### Шаг 1 — Создавать `Lock` лениво

```python
# БЫЛО:
class GlobalPolicyManager:
    _instance: GlobalPolicyManager | None = None
    _lock = asyncio.Lock()   # ← привязывается к loop при импорте
```

```python
# СТАЛО:
class GlobalPolicyManager:
    _instance: GlobalPolicyManager | None = None
    _lock: asyncio.Lock | None = None  # создаётся лениво

    @classmethod
    def _get_lock(cls) -> asyncio.Lock:
        """Получить или создать Lock в текущем event loop."""
        if cls._lock is None:
            cls._lock = asyncio.Lock()
        return cls._lock
```

### Шаг 2 — Использовать `_get_lock()` в `get_instance()`

```python
@classmethod
async def get_instance(
    cls, storage_path: Path | None = None
) -> GlobalPolicyManager:
    """Получить singleton instance. Thread-safe."""
    if cls._instance is not None:
        return cls._instance

    async with cls._get_lock():   # ← ленивая инициализация lock
        # Double-check после получения блокировки
        if cls._instance is not None:
            return cls._instance

        storage = GlobalPolicyStorage(storage_path=storage_path)
        instance = cls(storage)
        await instance.initialize()
        cls._instance = instance
        logger.debug("GlobalPolicyManager singleton created")
        return cls._instance
```

### Шаг 3 — Добавить метод сброса для тестов

```python
@classmethod
def reset_for_testing(cls) -> None:
    """Сбросить singleton состояние. Использовать ТОЛЬКО в тестах."""
    cls._instance = None
    cls._lock = None
    logger.debug("GlobalPolicyManager reset for testing")
```

### Шаг 4 — Добавить автоматический сброс в `conftest.py`

```python
# tests/conftest.py или tests/server/conftest.py:

import pytest
from codelab.server.protocol.handlers.global_policy_manager import GlobalPolicyManager


@pytest.fixture(autouse=True)
def reset_global_policy_manager():
    """Сбрасывать GlobalPolicyManager singleton перед каждым тестом."""
    GlobalPolicyManager.reset_for_testing()
    yield
    GlobalPolicyManager.reset_for_testing()
```

### Шаг 5 — Добавить защиту от использования вне event loop

```python
@classmethod
async def get_instance(cls, ...) -> GlobalPolicyManager:
    # Убедиться что мы находимся внутри running event loop
    try:
        asyncio.get_running_loop()
    except RuntimeError:
        raise RuntimeError(
            "GlobalPolicyManager.get_instance() must be called from within an async context"
        )
    ...
```

## Тесты

```python
import asyncio
import pytest
from codelab.server.protocol.handlers.global_policy_manager import GlobalPolicyManager


@pytest.fixture(autouse=True)
def reset_manager():
    GlobalPolicyManager.reset_for_testing()
    yield
    GlobalPolicyManager.reset_for_testing()


@pytest.mark.asyncio
async def test_get_instance_returns_singleton():
    instance1 = await GlobalPolicyManager.get_instance()
    instance2 = await GlobalPolicyManager.get_instance()
    assert instance1 is instance2


@pytest.mark.asyncio
async def test_reset_clears_singleton():
    instance1 = await GlobalPolicyManager.get_instance()
    GlobalPolicyManager.reset_for_testing()
    instance2 = await GlobalPolicyManager.get_instance()
    # После сброса создаётся новый экземпляр
    assert instance1 is not instance2


@pytest.mark.asyncio
async def test_concurrent_get_instance_creates_only_one():
    """Конкурентные вызовы должны вернуть один и тот же экземпляр."""
    results = await asyncio.gather(*[
        GlobalPolicyManager.get_instance()
        for _ in range(10)
    ])
    first = results[0]
    assert all(r is first for r in results)


@pytest.mark.asyncio
async def test_lock_works_in_new_event_loop():
    """Lock должен корректно работать после сброса в новом event loop."""
    GlobalPolicyManager.reset_for_testing()
    # Просто убеждаемся что не падает
    instance = await GlobalPolicyManager.get_instance()
    assert instance is not None
```
